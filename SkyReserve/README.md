# SkyReserve - Air Ticket Reservation System

A modern flight booking application built with **Angular 21**, featuring standalone components, Reactive Signals, and a premium UI powered by Tailwind CSS.

## Features

- **Interactive Flight Search**: Filterable search for routes across global destinations.
- **Signal-based State Management**: Reactive updates without complex boilerplate.
- **Dynamic Booking Flow**: Step-by-step passenger information and seat selection.
- **Digital Boarding Pass**: Instant e-ticket generation with printable layouts.
- **Authentication**: Secure (mocked) login and account management.
- **Booking History**: Persistent view of past reservations.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.

## Prerequisites

- Node.js (v20 or newer)
- npm (v10 or newer)
- Angular CLI (v19 or newer recommended)
